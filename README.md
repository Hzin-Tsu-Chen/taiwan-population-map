# 台灣各縣市人口指標互動地圖

串接內政部戶政司開放資料 API，分析全台 22 縣市人口指標，產出可切換指標的互動式地圖（GIS Choropleth 分級著色）。

## 功能

一張互動地圖，可切換三個指標，滑鼠移到縣市上顯示完整數據：

| 指標 | 說明 |
|---|---|
| 人口數 | 各縣市現住人口（2024）|
| 出生率（‰）| 每千人出生數 = 出生數 / 人口 × 1000 |
| 人口淨變化 | 2024 − 2023 人口，紅=流失 / 綠=增加 |

## 資料來源

全部為**內政部戶政司開放資料 API（2024 / 民國113 年）官方資料**：

- 人口數：ODRP019（村里級，程式加總至縣市）
- 出生數：ODRP055（鄉鎮市區級，程式加總至縣市）
- 地理界線：g0v 開放圖資

## 技術重點

- **ETL 分離設計**：`fetch_data.py` 負責從 API 抓取與計算，`map_analysis.py` 負責視覺化
- **空間運算**：計算人口密度前，先將經緯度轉為 TWD97 公尺座標系（EPSG:3826），因角度無法直接計算面積
- **資料清理**：2010 圖資的行政區劃更新至現況（桃園升格直轄市、官方正名「臺」）
- **資料驗證**：以新北市約 400 萬人口、全台總人口約 2340 萬做合理性檢查

## 主要發現

- 人口流失最多：臺北市（-20,953）、**彰化縣（-13,353，全台第 2）**
- 人口增加最多：桃園市（+21,189）、臺中市（+14,712）
- 出生率最高：雲林縣（10.52‰）；最低：基隆市（3.70‰）

## 如何執行

```bash
pip install geopandas folium pandas
python3 fetch_data.py      # 從政府 API 抓取資料 → indicators.csv
python3 map_analysis.py    # 產生互動地圖 → taiwan_map.html
```

## 檔案說明

| 檔案 | 說明 |
|---|---|
| `fetch_data.py` | ETL：串接政府 API、計算指標 |
| `map_analysis.py` | 讀地圖 + 指標、產生互動地圖 |
| `indicators.csv` | 計算好的各縣市指標 |
| `taiwan.geojson` | 縣市地理界線 |
| `taiwan_map.html` | 產出的互動地圖 |
